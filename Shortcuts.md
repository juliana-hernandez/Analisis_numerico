## Capitulo 1: Metodos de Bisección, regla falsa, punto fijo, Newton, Secante y  por lo menos uno de raíces múltiples, La interfaz debe graficar e imprimir la tabla solución en interfaz. Se debe poder incluir cualquier tipo de función algebráica y se tiene que tratar de ayudar al usuario cuando requiera calcular una derivada, así como explicar fácilmente como se ingresan los datos. También debe entregar un informe de ejecución y comparación en  todos los métodos ante un error específico, sea error relativo, absoluto o de condición, e identificar cual fue el mejor método de la ejecución. El informe es automático, pero el usuario puede elegir si correr o no el informe.

El informe es automático, pero el usuario puede elegir si correr o no el informe. El informe es automático, pero el usuario puede elegir si correr o no el informe.. Se debe poder incluir hasta 8 datos y se tiene que explicar fácilmente como se ingresan los datos.
